Этот скрипт позволяет переключать слайды, он автоматически подстраивается под количество слайдов, которые у него есть внутри,
а так же он адаптивен в плане JavaScript, то есть, может использоваться под различные устройста (мобильный телефон, ноутбук и т.д.),
так как в нем не указаны конретные величины в пикселях. Стили можно будет задать уже в CSS для своего проекта.


Использование:

2. JS скрипт:
	
	1) создается переменная slideIndex, которая будет определять какой слайд показывать;
	
	2) создаються переменные в которые помещаються элементы со страницы;
	
	3) вызываем фунцию с аргументом slideIndex, которая будет описана ниже, её можно вызвать заранее т.к. при объявлении будем использовать function declaration;
	
	4) создаем функцию showSlides(n) с аргуменнтом n (n-аргумент это номер слайда), это функция будет показывать нам нужный слайд и переключать класс активности
	у точек. В ней есть две проверки: первая проверка присваевает переменной slideIndex = 1, это сделано для того, что если при переключении на кнопку next
	закончаться слайды для переключения, далее будет реализован функионал, который будет автоматичеки показывать [нулевой] первый слайд на странице;
	вторая проверка присваевает переменной slideIndex slides.length; чтобы при переключении на кнопку prev, когда  закончаться слайды для переключения,
	далее будет реализован функионал, который будет автоматичеки показывать последний слайд на странице. Т.е. тем самым мы обезопасим себя и слайды будут
	всегда переключаться.
	Далее перебераем все слайды при помощи forEach и скрываем их, для точек делаем тоже самое.
	slides[slideIndex - 1].style.display = 'block' и dots[slideIndex - 1].classList.add('dot-active'), при первой загрузке
	будет показан первый [нулевой если по массиву js] слайд и добавлен класс dot-active первой кнопке, далее в завсимости от переменной slideIndex будет показан
	соответствующий слайд и присвоен класс активности для точки.
	
	5) создаем функцию plusSlides с аргументом n которая будет вызывать функцию showSlides с аргументом slideIndex += n, эта функция используется при клике на,
	кнопки prev и next, для увеличения(уменьшения) переменной slideIndex, с помощью которой определяетсы какой слайд показывать.

	6) создаем функцию currentSlide с аргументом n которая, так же будет вызывать функцию showSlides с аргументом slideIndex = n,эта функция используется
	при клике на элементы точки, для показа текущего слайда согласно нажатой точки.

	7) далее вещаем обработчики событий на кнопки prev и next которые вызывают функцию plusSlides с аргументом -1 для кнопки prev, что уменьшает переменную
	slideIndex и при помощи функции showSlides будет показан предыдущий слай, все тоже самое для кнопки next, только наоборот.

	8) далее вешаем обрабтчик событий на обертку в которой лежат все кнопки, чтобы использовать делегирование событий с помощью объекта event
	в коде указано как работает эта функция, но продублирую ниже с примером клика на третью точку на странице:
